# دليل المدير - تحديث القائمة

## 📱 كيفية تحديث القائمة (خطوات بسيطة)

---

## الخطوة 1️⃣: افتح لوحة التحكم

**على الكمبيوتر:**
```
http://localhost:5174/dashboard
```

**على الإنترنت (بعد النشر):**
```
https://rivera-menu.vercel.app/dashboard
```

---

## الخطوة 2️⃣: سجل الدخول

**كلمة المرور:**
```
rivera2026
```

⚠️ **مهم:** احفظ كلمة المرور في مكان آمن

---

## الخطوة 3️⃣: قم بالتعديلات

### ✅ لإضافة صنف جديد:

1. اذهب إلى تبويب **"أصناف القائمة"**
2. املأ النموذج على اليسار:

```
┌─────────────────────────────────┐
│ اسم الصنف (عربي) *             │ ← مطلوب
│ مثال: سلمون مشوي                │
├─────────────────────────────────┤
│ שם המנה (עברית)                │ ← اختياري
│ مثال: סלמון צלוי                │
├─────────────────────────────────┤
│ السعر *                         │ ← مطلوب
│ مثال: 85                        │
├─────────────────────────────────┤
│ الفئة *                         │ ← مطلوب
│ اختر من القائمة                 │
├─────────────────────────────────┤
│ اسم ملف الصورة                 │ ← مهم!
│ مثال: grilled-salmon.jpg       │
└─────────────────────────────────┘
```

3. انقر **"إضافة صنف"**

---

### ✏️ لتعديل صنف موجود:

1. ابحث عن الصنف في القائمة
2. انقر زر **"تعديل"**
3. عدّل المعلومات
4. انقر **"تحديث الصنف"**

---

### 🗑️ لحذف صنف:

1. ابحث عن الصنف
2. انقر زر **"حذف"**
3. أكّد الحذف

---

### 📁 لإضافة فئة جديدة:

1. اذهب إلى تبويب **"الفئات"**
2. املأ النموذج
3. انقر **"إضافة فئة"**

---

## الخطوة 4️⃣: عند إضافة صنف بصورة

### 🖼️ تسمية الصورة:

**✅ أسماء صحيحة:**
```
grilled-salmon.jpg
chocolate-cake.jpg
margherita-pizza.jpg
```

**❌ أسماء خاطئة:**
```
IMG_1234.jpg          ← غير واضح
صورة الطبق.jpg        ← عربي (لا يعمل)
my photo (2).jpg      ← مسافات وأقواس
```

### 📝 القاعدة:
- استخدم حروف إنجليزية صغيرة
- استخدم شرطة `-` بدلاً من المسافة
- أضف `.jpg` في النهاية
- تذكر الاسم بالضبط!

---

## الخطوة 5️⃣: تصدير البيانات

عندما تنتهي من جميع التعديلات:

1. انقر زر **"تصدير البيانات"** في الأعلى
2. سيتم تنزيل ملف مثل:
   ```
   menu-data-2026-02-14.json
   ```
3. احفظ الملف في مكان معروف

---

## الخطوة 6️⃣: أرسل للمبرمج

### 📧 عبر البريد الإلكتروني:

**إلى:** [بريد المبرمج]

**الموضوع:** تحديث القائمة - [التاريخ]

**المرفقات:**
```
✅ menu-data-2026-02-14.json
✅ grilled-salmon.jpg (إذا كان صنف جديد)
✅ chocolate-cake.jpg (إذا كان صنف جديد)
```

**نص الرسالة (مثال):**
```
مرحباً،

قمت بتحديث القائمة:

الأصناف الجديدة:
- سلمون مشوي (صورة مرفقة)
- كيك الشوكولاتة (صورة مرفقة)

تعديل الأسعار:
- بيتزا مارغريتا: من 45 إلى 48

حذف أصناف:
- طبق المعكرونة القديم

الرجاء التحديث عندما تستطيع.

شكراً!
```

---

### 💬 أو عبر WhatsApp/Telegram:

1. أرسل ملف JSON
2. أرسل جميع الصور الجديدة
3. اكتب قائمة بالتغييرات

---

## ✅ قائمة التحقق

قبل الإرسال، تأكد من:

```
□ قمت بجميع التعديلات المطلوبة
□ اختبرت القائمة (زر "عرض القائمة")
□ صدّرت ملف JSON
□ جمعت جميع الصور الجديدة
□ أسماء الصور تطابق ما كتبته في لوحة التحكم
□ أرسلت كل شيء للمبرمج
```

---

## ⏱️ المدة الزمنية

```
أنت ترسل التحديث
        ↓
5-10 دقائق: المبرمج يحدّث
        ↓
1-2 دقيقة: النشر التلقائي
        ↓
القائمة الجديدة تعمل! ✅
```

**المجموع: حوالي 10-15 دقيقة**

---

## 💡 نصائح مهمة

### ✅ افعل:
- صدّر JSON بعد كل جلسة عمل
- احتفظ بنسخة احتياطية من الصور
- اختبر التغييرات قبل التصدير
- استخدم أسماء واضحة للصور

### ❌ لا تفعل:
- لا تستخدم أحرف عربية في أسماء الصور
- لا تستخدم مسافات في أسماء الصور
- لا تنسَ إرفاق الصور مع JSON
- لا تعدّل ملف JSON يدوياً

---

## 🖼️ دليل الصور

### التقاط صور جيدة:

**✅ صورة جيدة:**
- إضاءة جيدة
- الطبق واضح
- خلفية نظيفة
- حجم معقول (أقل من 5 ميجا)

**❌ صورة سيئة:**
- مظلمة جداً
- غير واضحة
- حجم كبير جداً
- زاوية سيئة

---

## 🔧 حل المشاكل

### المشكلة: الصورة لا تظهر في القائمة

**الحل:**
1. تأكد أن اسم الصورة في لوحة التحكم يطابق اسم الملف بالضبط
2. تأكد أن الصورة بصيغة JPG أو PNG
3. تأكد أنك أرسلت الصورة للمبرمج

---

### المشكلة: نسيت اسم الصورة

**الحل:**
1. اذهب إلى تبويب "أصناف القائمة"
2. ابحث عن الصنف
3. انظر إلى حقل "اسم ملف الصورة"

---

### المشكلة: أريد تغيير صورة موجودة

**الحل:**
1. عدّل الصنف
2. غيّر اسم ملف الصورة إلى الاسم الجديد
3. صدّر JSON
4. أرسل الصورة الجديدة للمبرمج

---

## 📊 أمثلة عملية

### مثال 1: إضافة طبق جديد

```
1. اسم الصنف: سلمون مشوي
2. السعر: 85
3. الفئة: أسماك
4. اسم الصورة: grilled-salmon.jpg
5. التقط صورة للطبق
6. سمّ الصورة: grilled-salmon.jpg
7. صدّر JSON
8. أرسل JSON + grilled-salmon.jpg
```

---

### مثال 2: تعديل سعر

```
1. ابحث عن الصنف
2. انقر "تعديل"
3. غيّر السعر من 45 إلى 48
4. انقر "تحديث الصنف"
5. صدّر JSON
6. أرسل JSON (لا حاجة لصور)
```

---

### مثال 3: حذف صنف قديم

```
1. ابحث عن الصنف
2. انقر "حذف"
3. أكّد الحذف
4. صدّر JSON
5. أرسل JSON
```

## 🎯 أفضل الممارسات

### التحديثات المنتظمة:
- يمكنك التعديل في أي وقت
- أرسل التحديثات مرة يومياً (أو حسب الحاجة)
- اجمع عدة تغييرات معاً

### إدارة الصور:
- احتفظ بالصور الأصلية عالية الجودة
- سمّ الصور قبل التقاطها
- استخدم أسماء واضحة ووصفية

### التواصل:
- عنوان واضح للبريد
- اذكر ما تغيّر
- أكّد عندما تصبح القائمة حية
